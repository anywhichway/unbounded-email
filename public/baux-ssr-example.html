Baux Server-Side Rendering Example

How Modes Work in Baux Server:

  1. ssr-static: Server generates HTML without hydration script (for static content only)
  2. ssr-hydrate: Server generates HTML with hydration script for client-side interactivity

Note: The server version of Baux is optimized for server-side rendering and supports only these two modes. For client-side rendering, use baux-browser.js library.
